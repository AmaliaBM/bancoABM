# ABM BANK -
## English

### Description
This project is an academic assignment for the course **"Java and Database Management"**. Its main goal is to learn and practice **Java**, **IntelliJ IDEA**, **XAMPP**, and **MySQL**, applying object-oriented programming concepts, data persistence, and relational database management.

For simplicity and to focus on functionality, the security part has been simplified: passwords are stored in plain text, and no hashing or JWT is used. This allows focusing on exploring other features and flows of a banking application.

### Features
- Create users and bank accounts.
- Login with DNI.
- Check account balance.
- Make deposits and withdrawals.
- Transfer funds between accounts.
- View transaction history (deposits, withdrawals, and transfers) with date and time.
- Load data from **JSON** files.
- Persist data in MySQL using JDBC.

### Requirements
- **Java 17+**  
- **IntelliJ IDEA** (or other compatible IDE)  
- **XAMPP** or local MySQL server  
- `db.properties` configuration file for database connection.

> ⚠️ To run the application correctly, you must copy the `db.properties` file into the project structure.

### JSON Files
The project includes JSON files that allow loading initial data for users, accounts, and transactions, fulfilling a requirement that the application can read data from JSON files.

# Banco ABM – Proyecto Estudiantil

## Español

### Descripción
Este proyecto es un trabajo académico realizado como parte de la formación **"Java y Gestión de Base de Datos"**. Su objetivo principal es aprender y practicar el uso de **Java**, **IntelliJ IDEA**, **XAMPP** y **MySQL**, aplicando conceptos de programación orientada a objetos, persistencia de datos y manejo de bases de datos relacionales.

Por simplicidad y enfoque en la funcionalidad, la parte de seguridad está simplificada: las contraseñas se almacenan en texto plano y no se utiliza hashing ni JWT. Esto permite centrarse en aprender otras funcionalidades y flujos de una aplicación bancaria.

### Funcionalidades
- Crear usuarios y cuentas bancarias.
- Iniciar sesión con DNI.
- Consultar saldo de la cuenta.
- Realizar depósitos y retiros.
- Transferencias entre cuentas.
- Ver el historial de movimientos (depósitos, retiros y transferencias) con fecha y hora.
- Carga de datos desde archivos **JSON**.
- Persistencia de datos en MySQL mediante conexión JDBC.

### Requisitos
- **Java 17+**  
- **IntelliJ IDEA** (u otro IDE compatible)  
- **XAMPP** o servidor MySQL local  
- Archivo de configuración `db.properties` para la conexión a la base de datos.

> ⚠️ Para que la aplicación funcione correctamente, es obligatorio copiar el archivo `db.properties` en la estructura del proyecto.

### Archivos JSON
El proyecto incluye archivos JSON que permiten cargar los datos iniciales de usuarios, cuentas y movimientos, cumpliendo con una de las condiciones de entrega: que la aplicación pueda leer datos desde JSON.
